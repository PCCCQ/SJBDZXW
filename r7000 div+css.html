【联想拯救者R7000 2020测评：这次，联想和AMD联手拯救你的钱包】

在我刚开始接触 DIY 的时候，Intel在 CPU 这块拿捏得死死的，那时候装机选择酷睿 i 系列处理器是理所当然，广大DIY 玩家也已经习惯 AMD 的“日常翻身”了。所以今年大家都已经摇旗呐喊“AMD YES!”的时候，我对新锐龙的看法依然比较淡定。

直到我拿到联想拯救者 R7000 2020。

全新艺术设计，让操作轻松有趣

“暴打，简直就是暴打。”我几乎是拍着桌子看这台游戏本的配置。关键并不是同价位的AMD 怎么样暴打 Intel，而是它们压根就已经不是同一价位了。我节选了配置接近的几款拯救者，让大家先品品：

~R5 4600H + GTX 1650 + 16G +512G SSD，5199 元；
~R5 4600H + GTX 1650 Ti + 16G +512G SSD，5599 元；
~R7 4800H + GTX 1650 + 16G +512G SSD，5599 元；
~R7 4800H + GTX 1650 Ti + 16G +512G SSD，5999 元。

联想拯救者 Y7000 2020 部分配置及价格：

~i5-10300H + GTX 1650 + 16G + 512G SSD，5999 元；
~i5-10300H + GTX 1650 Ti + 16G + 512G SSD，6399元；
